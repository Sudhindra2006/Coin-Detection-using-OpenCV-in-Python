# Coin Detection using OpenCV

## Overview

This project detects coins in an image using OpenCV image processing techniques and blob detection.

## Requirements

* Python
* OpenCV
* NumPy
* Matplotlib

## Run the Project

1. Install required libraries
2. Open the notebook file
3. Run all cells step by step
4. View the detected coins in the output

## Output

The program detects coins from the input image and displays the total number of detected coins.
